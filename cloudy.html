What is this thing?

  • This weather app was built by Zoe during the Front-End Web Development course at General Assembly. It draws on various open source projects, credited below.
  • Live weather data is pulled in via the Forecast.io API.
  • The weather icon set is based on Adam Whitcroft's excellent Climacons. Other icons were taken from The Noun Project.
  • The location selector uses the JQuery UI Autocomplete widget.
